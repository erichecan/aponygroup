### 📋 抓取方案建议

#### 方案 1: 使用 Sitemap（推荐）⭐⭐⭐⭐⭐

**优势**:
- ✅ 有专门的新闻 sitemap: `sitemap/news.xml`
- ✅ 结构清晰，易于解析
- ✅ 包含所有新闻链接
- ✅ 符合网站官方规范

**实施步骤**:
1. 解析 sitemap/news.xml 获取新闻 URL 列表
2. 访问每个新闻详情页获取完整内容
3. 提取标题、内容、发布时间、分类等

#### 方案 2: 直接解析 HTML 列表页 ⭐⭐⭐⭐

**优势**:
- ✅ 可以获取最新的新闻列表
- ✅ 直接看到分类和摘要

**注意事项**:
- ⚠️ 需要解析 HTML 结构
- ⚠️ 网站改版可能导致解析失败
- ⚠️ 需要处理动态加载的内容

#### 方案 3: RSS Feed（需确认）⭐⭐⭐

需要检查是否有 RSS feed 可用

### 🚀 推荐实施方案

**建议采用 Sitemap 方式**，因为：
1. 更稳定（不会因页面改版失效）
2. 更高效（直接获取链接列表）
3. 更规范（使用官方提供的 sitemap）

### ⚠️ 注意事项

1. **请求频率限制**
   - 建议设置请求间隔（1-2秒/请求）
   - 避免对服务器造成压力

2. **内容过滤**
   - 只抓取与物流、跨境相关的新闻
   - 过滤掉完全不相关的内容

3. **版权声明**
   - 建议在新闻内容中添加来源标注
   - 例如："来源：出海网 (chwang.com)"

4. **错误处理**
   - 网站可能临时不可访问
   - 需要处理网络错误和超时

5. **内容更新**
   - 建议每天抓取一次（配合 Cloud Scheduler）
   - 只抓取最近 30 天的新闻

### 📝 配置示例

```javascript
{
  name: '出海网 - 跨境物流新闻',
  type: 'sitemap',
  url: 'https://www.chwang.com/sitemap/news.xml',
  enabled: true,  // 启用此源
  source: 'chwang',
  filter: true,   // 只抓取与物流相关的新闻
  maxItems: 20    // 最多抓取 20 条
}
```
