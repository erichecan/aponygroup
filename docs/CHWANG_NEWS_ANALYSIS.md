- **网站名称**: 出海网 - 企业跨境出海综合服务平台

### 内容相关性 ⭐⭐⭐⭐⭐
出海网的新闻内容与 AponyGroup 业务高度相关：

✅ **政策要闻** - 海关政策、贸易协定（对应 `policy` 分类）
- 如："美财长贝森特：无论法院判决是何结果，均不影响关税实施"
- 如："欧盟破获1.5亿元VAT欺诈走私案"

✅ **行业动态** - 物流、供应链、跨境（对应 `industry` 分类）
- 如："深圳跨境物流业频现爆雷"
- 如："Shopee优选仓启动地址变更"
- 如："万海503号货轮火灾后续"

✅ **公司新闻** - 物流企业动态（对应 `company` 分类）
- 如："京东收购德国零售巨头CECONOMY"

✅ **知识类** - 行业趋势、最佳实践（对应 `knowledge` 分类）
- 如："2025中国出海先锋TOP10榜单发布"

### 抓取方式建议

#### 方案 1: 使用 Sitemap（推荐）
出海网提供了专门的新闻 Sitemap：
- `https://www.chwang.com/sitemap/news.xml`

**优点**:
- 结构清晰，易于解析
- 包含所有新闻链接
- 符合网站官方规范

#### 方案 2: 解析 HTML 页面
从新闻列表页面直接解析：
- `https://www.chwang.com/news`

**优点**:
- 可以获取最新的新闻列表
- 可以直接看到分类和摘要

**缺点**:
- 需要解析 HTML 结构
- 网站改版可能导致解析失败

#### 方案 3: RSS Feed（需验证）
需要检查是否有 RSS feed 可用

## ⚠️ 注意事项

1. **请求频率限制**
   - 建议设置请求间隔（如每次请求间隔 1-2 秒）
   - 避免对服务器造成压力

2. **内容过滤**
   - 只抓取与物流、跨境相关的新闻
   - 过滤掉完全不相关的内容

3. **版权声明**
   - 建议在新闻内容中添加来源标注
   - 如："来源：出海网"

4. **错误处理**
   - 网站可能临时不可访问
   - 需要处理网络错误和超时

5. **内容更新**
   - 建议每天抓取一次（配合 Cloud Scheduler）
   - 只抓取最近 30 天的新闻
